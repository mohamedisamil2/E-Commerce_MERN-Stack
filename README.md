# 🛒 MERN E-Commerce

A Full-Stack E-Commerce platform built with the MERN Stack.

## ✨ Features

- User Authentication (JWT + Cookies)
- Product Listing
- Product Search
- Shopping Cart
- Checkout
- Orders Management
- Responsive Design

## 🛠️ Tech Stack

### Frontend
- React.js
- Redux Toolkit
- RTK Query
- React Router
- Tailwind CSS
- DaisyUI
- React Hook Form
- Zod

### Backend
- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT
- Cookie Parser

## 📂 Project Structure

```text
frontend/
backend/
```

## 🚀 Installation

### Clone repository

```bash
git clone https://github.com/mohamedisamil2/E-Commerce_MERN_Stack.git
```

### Install dependencies

```bash
cd frontend
pnpm add

cd ../backend
pn add
```

### Run project

```bash
pn run dev
```
